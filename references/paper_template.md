# 小红书论文分享文案模板

## 模板一：综述类论文

```
{emoji_intro}

{intro_sentence}

📕《{paper_title}》
这篇文章全面综述了{core_topic}的核心方法、理论基础与前沿进展。文章系统梳理了{method_categories}，并深入探讨了{key_challenges}等问题。此外，文章还重点分析了{future_directions}，是理解{domain}体系与前沿研究的指南之一。

{tags}
```

**变量说明**：
- `{emoji_intro}`: 标题党引入
- `{intro_sentence}`: 开头句
- `{paper_title}`: 论文标题
- `{core_topic}`: 核心主题
- `{method_categories}`: 方法分类
- `{key_challenges}`: 关键挑战
- `{future_directions}`: 未来方向
- `{domain}`: 领域名称
- `{tags}`: 话题标签

## 标题模板

### 综述类

1. "分享{N}篇{topic}综述，{domain}入门推荐！"
2. "这篇{topic}综述太全了！{domain}必看"
3. "{topic}领域全景解读：从{A}到{B}"

### 方法类

1. "{topic}新突破！{method}效果提升{N}%"
2. "这篇论文改变了{domain}：{short_title}"
3. "{method}：解决{problem}的新思路"

### 应用类

1. "{method}在{domain}的应用：{result}"
2. "从论文到落地：{topic}实战分享"
3. "{domain} + {method}：这个方向要火了"

## 使用建议

1. **模板是起点**：根据具体论文灵活调整
2. **保持个性**：加入自己的理解和见解
3. **控制长度**：正文 300-500 字为宜
4. **突出重点**：用粗体、emoji 标记关键信息
5. **引导互动**：结尾提问或号召行动

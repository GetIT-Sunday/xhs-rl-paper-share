# 小红书爆款文案风格指南

## 标题原则

**核心：吸引眼球 + 传递价值**

### ✅ 好的标题

1. **数字开头**
   - "分享3个RL技巧，模型效果提升50%"
   - "这篇综述总结了10种主流RL算法"

2. **疑问句/反问句**
   - "强化学习还能这么用？"
   - "具身智能+RL会改变什么？"

3. **情绪化表达**
   - "这篇论文太重要了！"
   - "RL领域重磅综述！"

4. **对比/冲突**
   - "传统方法 vs 深度RL：谁更胜一筹？"
   - "从入门到精通：RL学习路线图"

### ❌ 不好的标题

- "Reinforcement Learning: An Overview 论文解读"
- "强化学习方法综述"
- "一篇论文分享"

---

## 正文结构

**三段式结构：引入 + 内容 + 互动**

### 1. 引入（1-2句）

建立共鸣，引起兴趣：

```
从最新 arXiv 论文中挖到一篇好文，来分享一下！
```

```
最近在读强化学习相关论文，这篇很有价值：
```

### 2. 内容（主体）

**使用 emoji 分段**：

- 📕 重要论文
- 📗 方法介绍
- 📘 实验结果

**核心观点强调**（小红书不渲染 markdown，用【】替代 ** **）：

```
文章系统梳理了【基于价值】、【基于策略】、【基于模型】三大类RL方法
```

**避免堆砌术语**：

❌ "本文提出了基于actor-critic架构的model-based RL算法，通过学习world model实现sample-efficient policy optimization"

✅ "这篇文章提出了一种新方法，通过学习环境模型来加速强化学习训练，让AI能更快学会新任务"

### 3. 互动（结尾）

引导评论、点赞、收藏：

```
你对强化学习的哪个方向最感兴趣？评论区聊聊～
```

```
觉得有用的话记得点赞收藏哦！
```

## 语言风格

### 口语化

❌ "该研究提出了一种新颖的方法论框架"

✅ "这篇文章提出了一个新方法"

❌ "实验结果表明所提方法在多个基准测试中取得了显著提升"

✅ "实验结果显示，这个方法在多个任务上都表现更好"

### 亲切感

使用"我"、"我们"、"你"：

```
我最近在读这篇论文，发现...
```

```
这篇文章帮你理清RL的核心概念
```

### 专业与通俗平衡

**专业**：保证准确性
**通俗**：降低理解门槛

```
这篇文章全面综述了强化学习的核心方法（如Q-learning、PPO、SAC等）
```

既有专业术语，又有具体例子。

## 常见问题

### Q: 标题太长怎么办？

A: 控制在 20-30 字以内，使用省略号：

```
这篇RL综述把PPO/SAC/DQN讲透了...
```

### Q: 如何避免重复？

A: 维护已发布论文列表，发布前检查：

```bash
python3 scripts/publish_to_xhs.py --mark-published
```

### Q: 文案生成质量不好？

A: 手动修改生成的文案，或者调整模板：

编辑 `scripts/generate_content.py` 中的模板和关键词。
